# flet-gpt-markdown
FletGptMarkdown control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-gpt-markdown @ git+https://github.com/MyGithubAccount/flet-gpt-markdown",
  "flet>=0.84.0",
]
```

* **uv/pip dependency**

If the package is published on pypi.org:

```
dependencies = [
  "flet-gpt-markdown",
  "flet>=0.84.0",
]
```

Build your app:
```
flet build macos -v
```
